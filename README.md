# Rhgf (Gestión financiera de recursos humanos).
El paquete Rhgf (Gestión financiera de recursos humanos) de V12 es el módulo que integra la nómina y la gestión financiera del personal, ofreciendo además una amplia gama de aplicaciones como el acceso de empleados. Su propósito es administrar de manera eficiente los pagos, prestaciones y obligaciones fiscales, mientras brinda a los trabajadores herramientas de autogestión dentro del ERP.
